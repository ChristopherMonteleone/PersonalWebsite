Cannon Clash

Made with Unreal Engine 5

View on GitHub

Gif or Video Gameplay



Description

Cannon Clash offers an engaging and dynamic experience as a top-down shooter game that captivates players with its responsive gameplay mechanics. It features a full game loop with scaling difficulty and an engaging upgrade system for immersive play. The game's world is brought to life using Unreal Engine's Procedural Generation tools and enhanced by the Niagara system's captivating visual effects. Its architecture is built on a modular codebase that adheres to Unreal Engine 5's coding standards and Object-Oriented Design Principles for efficient performance and easy expandability. Cannon Clash offers a compelling blend of technical proficiency and entertaining gameplay.

Background

Cannon Clash serves as an evolution of the "Toon Tanks V2" project, initially introduced by the esteemed GameDev.tv team in their comprehensive "Unreal Engine 5 C++ Developer Course". This course laid the foundational blocks of game development principles from which Cannon Clash was crafted. Leveraging the core concepts acquired from the course, I have expanded upon the original framework to forge Cannon Clash into a distinct and enriched experience. I believe that this project howcases a commitment to high-quality design principles. This dedication has culminated in a game characterized by its scalability and a meticulously organized codebase, ensuring a seamless and immersive gameplay experience.

Astroponics

Made with Godot

View on GitHub



Description

Astroponics is an engaging and educational game focused on hydroponic farming. This project aims to provide users with a hands-on experience in the innovative world of hydroponics. Players will navigate a futuristic spaceship environment, learning the nuances of planting, nurturing, and harvesting various plant varieties associated with hydroponics.

Background

Astroponics was developed as a senior group project for the University of Nevada, Las Vegas. It consisted of seven individuals working as a team over twelve weeks. This team coordinated through weekly meetings that discussed the planning and execution of tasks in a collaborative manner, using the GDScript language, Godot game engine, and Git for version control. Unit testing was done using the Godot Unit Tests plugin. I believe that it demonstrates my ability to work in a team effectively to deliver an end result.
